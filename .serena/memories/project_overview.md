# TradingAgents 项目概述

## 项目目的
TradingAgents 是一个多智能体金融交易框架，模拟真实交易公司的运作模式。通过部署专业的LLM驱动智能体（基本面分析师、情绪专家、技术分析师、交易员、风险管理团队），平台协作评估市场条件并做出交易决策。

## 核心特性
- 多智能体协作决策系统
- 专业化角色分工（分析师团队、研究团队、交易员、风险管理）
- 动态辩论机制优化策略
- 实时数据获取和历史数据回测
- 模块化架构设计

## 业务领域
金融量化交易、投资决策支持、风险管理

## 适用场景
- 量化交易策略开发
- 投资决策辅助
- 金融市场研究
- 风险评估和管理
